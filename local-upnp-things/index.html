Local UPnP Things

Local UPnP Things is a local Wi-Fi/LAN controller for compatible local UPnP/SOAP Things.

Local-first behavior

  • Find compatible Things on the same Wi-Fi/LAN.
  • Read local device descriptions and save known Things on device.
  • Refresh state and control compatible local on/off services when available.
  • No cloud account is required.
